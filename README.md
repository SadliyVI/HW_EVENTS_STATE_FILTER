# Домашнее задание к занятию React. События и Состояние

## HW_EVENTS_STATE_FILTER — (Vite + React)

Проект на **Vite + React**, демонстрирующий веб-приложение для отображения портфолио проектов с возможностью их фильтрации по категориям

### Требования

- React
- Vite
- Node.js
- npm

### Установка и запуск

1) Установить зависимости:

npm install

2) Запустить dev-сервер:

npm run dev

Откройте адрес, который выведет Vite в терминале (обычно <http://localhost:5173/>)

3) Сборка и предпросмотр:

Сборка:

npm run build

Предпросмотр собранного проекта:

npm run preview
